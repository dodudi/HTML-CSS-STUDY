flex 레이아웃

display

flex로 지정을 하면 flex방식으로 자식 요소들을 배치한다.
flex와 inline-flex가 있다

flex-direction

아이템들이 배치되는 축의 방향을 결정하는 속성입니다.
row(가로로), column(세로로), row-reverse(가로역순), column-reverse(세로역순)가 있다.

flex-wrap

내부 요소가 컨테이너를 넘어갈 때 처리 방식을 정합니다.
nowrap(삐쳐나옴), wrap(아래로), wrap-reverse가 있다.

flex-flow

flex-direction과 felx-wrap을 한번에 설정가능한 속성.
-> flex-flow: row wrap 등으로 사용한다.

justify-content

메인축 방향 정렬입니다.
flex-start(시작점), flex-end(끝점), center(중앙), space-between(아이템 사이에 균일한 간격), space-around(아이템 양 옆으로), space-evenly(아이템 양 옆으로 균일)가 있다

align-items

수직축 방향 정렬
stretch(수직축 방향으로 끝까지 늘어남), flex-start(시작점), flex-end(끝점), center(가운데), baseline(텍스트 베이스라인 기준으로)가 있다.

align-content

flex-wrap: wrap이 설정된 상태에서, 아이템들의 행이 2줄 이상 되었을 때의 수직축 방향 정렬을 결정하는 속성입니다.
stretch, flex-start, flex-end, center, space-between, space-around, space-evenly

flex-basis

flex 아이템의 기본 크기 설정. auto, 0, 50%, 300px, 10rem, content

flex-grow

아이템이 felx-basis의 값보다 커질 수 있는지 설정하는 속성
0보다 큰 값이면 아이템이 유연한 박스로 변하고 원래의 크기보다 커지며 빈공간을 메운다. 1:2:1 등등 비율로 설정

flex-shrink

flex-grow와 쌍을 이루는 속성으로, 아이템이 flex-basis의 값보다 작아질 수 있는지 결정합니다.
얼마든 일단 0보다 큰 값이 세팅이 되면 해당 아이템이 유연한 박스로 변하고 flex-basis보다 작아진다.
0으로 설정되면 아이템의 크기가 flex-basis보다 작아지지 않아.
기본값이 1이다.

flex

flex-grow, flex-shrink, flex-basis를 한 번에 사용 가능한 속성
felx: 1 1 auto; -> felx-grow: 1; flex-shrink:1; flex-basis:auto

align-self

align-items의 아이템 버전. align-items가 전체 아이템의 수직축 정렬이면, 이것은 해당 아이템의 수직충 방향 정렬
auto, stretch, flex-start, flex-end, center, baseline

order

아이템들의 나열 순서 결정하는 속성
작은 숫자일 수록 먼저 배치.

z-index

z축 정렬을 할 수 있다 숫자가 클수록 위로 올라간다.
gap속성으로는 아이템 간에 간격을 줄 수 있으며 두 개의 값을 넣어서
가로 간격과 세로간격을 다르게 지정하는것도 가능하다.
