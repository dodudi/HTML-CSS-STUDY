h1 태그

제목 태그입니다.

p 태그

p태그는 문단을 나타냅니다.
p태그 마다 각각 줄바꿈이 됩니다.

이 문장은 p 태그 안에 있습니다.

hr 태그

hr태그는 줄을 그어줍니다.

b 태그

b태그는 글자가 굵어집니다. 이 문장은 b 태그 안에 있습니다.

strong 태그

strong 태그는 글자가 굵어집니다.
강조할 때 사용하면 좋습니다.

i 태그

i 태그는 글자를 기울입니다. 이 문장은 i 태그 안에 있습니다.

em 태그

em 태그는 글자를 기울임니다.
강조할 떄 사용합니다.
이 문장은 em 태그 안에 있습니다.

q 태그

짧은 인용문에 사용합니다.
cite를 속성으로 쓸때 인용한 사이트 주소를 적어줍니다.

blockquote 태그

blockqute 태그는 긴 인용문에 사용합니다.
cite 속성이 있습니다.

mark 태그

mark 태그는 사용자의 행동과 관련 있는 부분에 사용된다.
예를 들어 검색 결과, 인용문에서 주시해야할 부분, css와 병행하여 사용합니다.

ul태그

ul태그는 li와 함께 사용합니다.
목록을 나타내줍니다.

ol태그

ol태그는 li와 함께 사용합니다.
목록을 나타내줍니다.

dd, dt, dl 태그

dl 안에 dt, dd를 정의한다.
dt는 제목이며 dd는 목록입니다.

img 태그

src ="이미지 주소"
alt ="이미지가 나오지 않을때"
title ="마우스 올렸을 때 말"

table 태그

tr -> (th, td)를 사용한다.
th, td는 colspan과 rowspan로 행과 열을 병합할수 있다.

catpion 태그

table태그안에 사용가능.
범례.

thead, tbody, tfoot 태그

table 태그 안에 사용가능.
thead -> tr -> (tr,td)

colgroup 태그

table 태그 안에 사용가능.

a 태그

링크 가능하다.
href = "링크"
target = "_self"(_blank, _parent, _top 등)
id를 이용해서 원하는 곳으로 이동하는것도 가능하다.(href="#id")

address 태그

웹사이트 주소: 나이스 주소
전화 이메일
